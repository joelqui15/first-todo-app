# TripleTen Sprint 4–5 Mini Projects

This repository contains mini projects built to reinforce Sprint 4 and 5 concepts:

## Covered Topics

### Sprint 4

- Variables and data types
- Conditionals and loops
- Functions and scope
- Arrays and objects
- Basic DOM manipulation
- Event listeners

### Sprint 5

- Array methods (map, filter, reduce, sort, find, some, every)
- Safe DOM updates (textContent vs innerHTML)
- createElement / cloneNode
- Templates
- Removing and moving elements
- DOM family traversal (parentElement, closest, children, siblings)

---

## Projects

### 01 – Safe Comments Feed

A dynamic comment system with:

- Template cloning
- Safe user input handling
- Like and delete functionality
- Sorting
- Statistics powered by array methods

## How To Run

Open any project's `index.html` file directly in the browser.
